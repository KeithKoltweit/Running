jQuery and Forms & Validation

I have javascript on my Contact Us page to validate information the user types in or doesn't put in. If a field if required and the user doesn't supply that information and clicks the submit button a message will appear in red telling them that they missed a that field with an appropriate message on the top of the form. I also, decided to use jQuery to show the user what field they are on with focus that changes the background color. Then if the user clicks away from a required field I have blur property that changes the background color to red, showing that they need to fill out that field.
